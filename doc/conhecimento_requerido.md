# Conhecimentos requeridos.

Para trabalhar com Adianti Framework é recomendavel que tenha conhecimento das tecnologias abaixo
* PHP - exigido 
* PHP com POO - desejável 
* HTML 5 - desejável 
* JavaScript - desejável
